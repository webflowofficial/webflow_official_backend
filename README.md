
# WebFlow 

Official Page of an amazing website development company


## Run Locally

Clone the project

```bash
  git clone https://github.com/webflowofficial/webflow_official_backend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
